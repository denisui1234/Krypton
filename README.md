
# Krypton

A simple script that downloads a debian proot in termux and sets it up with common server utilities.



## Includes

- Cloudflared
- Tmux
- Apache2
- Mediamtx
- Ffmpeg
- Zrok 1
- Openssh

## Configure dependencies



```bash
  pkg update
  apt install wget
```
